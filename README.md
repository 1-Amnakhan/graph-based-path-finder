\# Graph-Based Path Finder



A Path finder that models and analyzes a weighted directed graph. The project allows users to create custom graphs, explore connections between nodes, find shortest paths, and analyze graph properties through an interactive menu-driven interface.



\## Features



\- Create graphs with custom node names (e.g., city names)

\- Add and display graph edges

\- Perform Breadth-First Search (BFS)

\- Perform Depth-First Search (DFS)

\- Find shortest paths using Dijkstra's Algorithm

\- Check graph connectivity

\- Detect cycles in the graph

\- Input validation and error handling



\## Technologies Used



\- C++

\- Object-Oriented Programming (OOP)

\- STL (Vector, Queue, Priority Queue, Unordered Map)

\- Graph Data Structures and Algorithms



\## Algorithms Implemented



\- Breadth-First Search (BFS)

\- Depth-First Search (DFS)

\- Dijkstra's Shortest Path Algorithm

\- DFS-Based Cycle Detection

\- Connectivity Analysis



\## How to Run



Compile the program:



```bash

g++ main.cpp -o graph

```



Run the executable:



```bash

./graph

```
